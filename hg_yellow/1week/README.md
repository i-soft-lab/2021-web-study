# 1주차 정리

## 인터넷이 동작하는 원리
인터넷이 동작하기 위해서는 필요한 컴퓨터 : 2대
- 웹 브라우저가 깔린 컴퓨터
- 웹 서버가 깔린 컴퓨터<br/>

웹 브라우저 : HTML 문서와 그림, 멀티미디어 파일등 월드 와이드 웹을 기반으로 한 인터넷의 컨텐츠를 검색 및 열람하기 위한 응용 프로그램의 총칭.
- 통칭 : 클라이언트
- Ex) 크롬 사파리, 윈도우 익스플로어

웹서버 : 웹 브라우저와 같은 클라이언트로부터 HTTP 요청을 받아들이고, HTML 문서와 같은 웹 페이지를 반환하는 컴퓨터 프로그램
- 통칭 : 서버
- Ex) 아파치 HTTP 서버, 마이크로소프트 인터넷 정보 서비스, 톰캣

웹 브라우저가 작동하는 과정 : 웹서버에 필요한 정보를 입력
- info.cern.ch라는 서버에 index.html라는 정보를 불러오기 -> http://info.cern.ch/index.html

웹서버가 작동하는 과정 : 웹브라우저에서 요청한 정보를 웹브라우저에 응답
- 클라이언트가 index.html을 요청하면 디렉토리 안에서 index.html의 파일을 찾아서 요청한 클라이언트에게 응답
